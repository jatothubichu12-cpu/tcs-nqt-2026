# 🎯 TCS NQT 2026 — 90-Day Preparation Tracker

A Progressive Web App (PWA) to track your complete 3-month TCS NQT preparation journey — from Written Test to ILP.

## 📋 What's Included

| Month | Focus | Days |
|-------|-------|------|
| Month 1 | Aptitude + Verbal + Reasoning + Programming Logic | Day 1–30 |
| Month 2 | Coding + OOPS + DBMS + OS + CN + DSA + Projects | Day 31–60 |
| Month 3 | TR + MR + HR Interview Prep + ILP (Java/SQL/Linux/Agile) | Day 61–90 |

## 🚀 How to Use

1. **Download** `index.html` and `manifest.json`
2. **Open** `index.html` in your browser (Chrome/Edge recommended)
3. **Check off** each day as you complete it
4. Progress is **auto-saved** in your browser's localStorage
5. **Export JSON** to save/commit your progress to GitHub

## 📱 Add as PWA (Mobile)

1. Open `index.html` in Chrome mobile
2. Tap "Add to Home Screen"
3. Use like a native app, offline!

## 📂 Host on GitHub Pages

1. Create a new GitHub repo
2. Upload `index.html` and `manifest.json`
3. Go to **Settings → Pages → Deploy from branch (main)**
4. Access at `https://yourusername.github.io/reponame`
